Frequently Asked Questions

var: Function-scoped, can be redeclared and updated.

let: Block-scoped, can be updated but not redeclared.

const: Block-scoped, cannot be updated or redeclared.

map(): Returns a new array after applying a function to each element.

forEach(): Executes a function for each element but does not return anything.

filter(): Returns a new array with elements that pass a condition.

Arrow functions have a shorter syntax and do not have their own this.

Example:

const add = (a, b) => a + b;

A Promise represents an asynchronous operation that may resolve or reject in the future.

It has three states: pending, fulfilled, and rejected.

Closures allow a function to access variables from its outer scope, even after the outer function has finished executing.

This helps preserve state in asynchronous programming or callbacks.
